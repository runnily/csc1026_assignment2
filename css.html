CSS: Style

Now you’ve covered html, we move on to CSS. The first idea of CSS came from Hakon Wium Lie while working CERN. For now, without CSS our webpages is looking boring. We need some additional organization and aesthetic to enhance the users experience. We can achieve this using CSS. Before we rush into anything its important to understand the balance of information and style. For example, if we use no style this would not influence users to go on our site. However, if over styled our website it would draw our user away. The style you use also depends on the purpose of the website; A website for art purposes would provide more style.
Cascaded Style sheets specify formatting rules. These rules consist of selectors and declarations of proprieties. For example, below is a HTML code for a paragraph <p> This is a paragraph should have white text </p> If we then want to specify the style of the paragraph, we would do this using CSS. Using the selector P, and declarations to define its properties. Below is a simple example of using CSS to apply a style to the element p {background-color: blue; color: white;} This basic idea of CSS is specifying how the elements looks by using selectors and declarations.

CSS: More on Style

Multiple selectors can be used with comma to seperate indivdual tags. Below shows an example h1, h2, h3, h4 { background-color: white; color: blue; font-style: italic; } Going back to the html document, we want a way to link the html to the CSS. There are many ways in which you could do this. One way you could do is to add the CSS within the HTML. This is known as an internal style sheet. Here we set out the style of the CSS within the <style> tag of the html. This style tags are contained with the <head> section of the page. <head> <title> Internal Example </title> <style type =”text/css”> H1 {color: green Font-style: italic;} </style> Another method is using external style sheets, these are where the style sheet is stored in a separate file. Therefore, the html document only contains html code, which links to an external CSS style sheet. An example is shown below: <head> <title>CSS example </title> <link rel = “stylesheet” , type=”text/css” href=”mystyle.css”/> </head> An alternative way to include external style sheets could be done using the “@import”. <head> <title> CSS import example </title> <tyle type=”text/css”> @import = url(style.css) </style> </head> When we separate the content and apperance were doing something called two degrees of separation. This is more practical when working with a webpage that consist of many pages, as we could simply change one line of code through the CSS rather than going through all our html code. Where we want to apply a style to a specific element, on a particular page; we use an inline tag. This is done within html in which we specify the style attribute. An example is shown below: <p style= “color:white; background-color: blue;”> Hello</p>

CSS selectors

CSS selectors are used to set a style an element within the html document. They are three basic selectors to define the appearance of these elements within the html. These include tags selectors, if we specify a tag in the CSS, this tells the browser to render all instances of the tag within html to the property appearance specified in the CSS. For example if we were to use the selector p { /*specified property here*/} in CSS, all the instances of <p> tags within html would style with the specified property defined in the CSS. Another selector is the Class selector, this works the same way, by matching all the tags in that class to be styled to the specified property set in CSS. For example <p class= "special" </p> and <h1 class= "special" /> . Both are different tags, but each tag has the same class. This class would be addressed and set to a specific style using CSS. For example: .special { /*specified appearance*/. The last selector is the id selector. This uniquely identifys one tags, to style in the CSS. For example: <p> <span> id=”oneoff”> Important </span> </p>. This individual tag can be styled using CSS, like this: #oneoff {color: red;} </p>

CSS units

CSS is able to support many types of measurement unit. There are two types of unit of measurements we can use in CSS. First one is an absolute unit. These units of measurements are calculated independently of another page. This is useful when you want the measurement to be precise. Examples include; pixels (px) points (pt) and millimetres (mm). The second units are relative units. These units are calculated proportionally against another page content or browser default. Example of units include, percentages (%), Ems (em) and Exes (ex). A good designed website will use a combination of both.

The CSS box Model

The CSS box model is important in terms of the layout structure. The box model is more focused on where the box (tag) is placed rather than the contents within the box. We vislise each tag (links, paragraphs) as a box. Therefore each tag corresponds to a box, and each box has a hierarchy structure. This means the properties of the boxes are not inherited from the parent boxes. The box properties define where the box is positioned and the appearance of the box. The are three main box properties. One is called the border. The border is the outline of the box, and there are 3 properties of the border we could set. This is the width of the border: border-width: 20px; The style of the border, this sets if the border is dotted etc border-style: sold|dashed| The color of the border, a valid colour value border-color: blue; Below is a simple shorthand we can set the 3 properties of the border: p {border: 20px solid geen } Another box property is called padding. This is the space between the inside of the border and outside of the contents. This creates inside of the boxes. p {border: 20px solid green; padding: 20px;} The last box property is called the margin, this is the space outside of the border of the box. This is good for content spacing. p {border: 20px solid green; padding: 20px; margin: 20px;}

Declaration Results

border-left-width: 20px; width 20px on left-hand side only

The position property specifies how boxes are positioned using CSS

Type of Positions Description
static A default position for boxes. This is not normally used unless to override another position
relative An element with the position absolute is position relatively the nearest positioned box
Absolute This removes elements from natural page flow and allows it to be positioned anywhere
Fixed As the name suggest, it sets a box to a fix position in the page

From here you can use the properties top and bottom to precisely position your elements on the page
